# @jsonic/json5

This plugin configures the [Jsonic](https://jsonic.senecajs.org) JSON parser to parse JSON5 syntax.

This repository contains:

| Path | Description |
|---|---|
| [`ts/`](ts/) | TypeScript / JavaScript implementation. |
| [`go/`](go/) | Go port. |

See [`ts/README.md`](ts/README.md) for usage.
