# @jsonic/json5 — Go

A [Jsonic](https://github.com/jsonicjs/jsonic) plugin for parsing
[JSON5](https://json5.org).

```bash
go get github.com/jsonicjs/json5/go
```

See the full
[Go documentation](https://github.com/jsonicjs/json5/blob/main/doc/json5-go.md)
(tutorials, how-to guides, explanation, reference) at the top of the
repository.
